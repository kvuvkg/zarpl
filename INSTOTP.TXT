
      Работа пользователя начинается с вызова программы  на выполнение .
   На экране появляется головное меню:
  ┌───────────────────────────────────────┐
  │ Сервис   Отпуск   Больничные   Выxод  │
  └───────────────────────────────────────┘
    При входе в режим "Сервис" высвечивается след подменю:
        Дата расчета
        Накопление
        Чистка базы
        Минимальная З/пл
        Запись не выданной з/пл
     
               Режим " Дата расчета".
   На экране высвечивается дата за которую будет производится расчет
  (отпускных и бол листов). При неодходимости ее надо скорректировать.

          Дата расчета    15.07.1996
 
              Режим " Накопление".
      Данный режим выполняется в том случае, когда не было произведено 
   перехода на новый месяц.Его необходимо производить после получения
   и проверке всех выходных форм.
      Если переход на новый месяц произведен данный пункт меню
   нельзя выполнять!!!!!!!!!!
 
    Режим " Минимальная З/пл".

    На экране высвечивается MIN зароботная плата при ее изменении ее необходимо
  корректировать!!!!!!!!!!

          Минимальная з/п      1500000.00
      
       При входе в режим  "Отпуск" высвечивается след подменю:

       │Ввод отпускных    │
       │Расчет            │
       │Печать плат отпуск│
       │Запись информации │
       │Справ коэф отпуска│
       └──────────────────┘

            При входе в режим "Ввод отпускных"
   на экране появляется след окно:
     ┌─[■]──────────────────────────────────────----
     │Чис│Цех │Табел.│  Дата    │К-во│приз│приз│приз
     │ло │    │номер │  отпуска │дней│опт.│вып.│рас.
     │──────────────────────────────────────────────
     │  1    1      1 15.08.1996   24    0    0    1
     │  1    1      2 21.08.1996   24    0    0    1
     │  1    1      3 21.08.1996   23    0    0    1


     Где  число означает день выплаты отрускных в кассе.
          Дата отпуска- дата начала отпуска.
          К-во дней - количество дней отпуска. 
          приз опт.  0-очередной отпуск 1- ком за неиспользованный отпуск .
          приз вып   0- получение денег в кассе 1-в заработанную плату.
          приз рас.  1-означает что отпускные расчитаны 0 не расчитаны .

      Режим "Расчет" необходимо производить после ввода отпускных записок.
   В данном режиме происходит расчет отпускных записок и запись в промежуточный
    файл   

           " Печать плат отпуск".
   В данном режиме работы происходит печать платежной ведомости на выдачу
   отпускных в кассу (принтер должен находится в режиме готов).

     После получения всех плат ведомостей  и перхода на новый месяц!!!!!!!
   выполняется режим  " Запись информации". В данном режиме происходит перевод
  расчитанных отпускных в лицевые счета .Далее все исправления делаются
  только в лицевом счете (необходимо помнить , что данный 
  режим выполняется только 1 раз!!!!!. В противном случае произойдет удвоение
  информации .
         Чистка базы.
   Данный режим работы производится после выполнения  режима " Запись информации"
  Вся информация по вводу и расчету отпускных и бол будет уничтожена. И базы
  подготовлены для ввода и расчета на новый месяц .


      Справ коэф отпуска
  При входе в данный режим работы на экране высвечивается след окно
    ┌─[■]────────────────────-----
    │       Период        │Коэф-
    │    с     │    по    │фиент.
    │────────────────────────────
    │01.02.1996 01.09.1996  2.000

   Где период означает действие коф т. е. если сумма за месяц для расчета
  отпускных попадает в период, то сумма умножается на коэф .
     
          При входе в режим  "Больничные".
      Hа экране высвечивается след подменю:

     ┌─────────────────────┐
     │Ввод больничных      │
     │Запись информации    │
     │Справ коэф больничных│
     └─────────────────────┘
         При входе в под меню "Ввод больничных"
    на экране высвечивается след окно:

      ┌─[■]───────────────────────────────────────────────────---─┐
      │   дата   │  номер │цех │таб.  │   дата   │к-во│ % │вид│гр.│
      │  р. БЛ   │   БЛ   │    │номер │    БЛ    │дней│опл│рас│бол│
      │───────────────────────────────────────────────────────────│
      │15.07.1996      123    2     40 15.07.1996    2 100   1   0│
      │                                                           │
                                                           
     дата р. БЛ -это дата, с которой начинается выборка начислений
   для расчета бол листов (на пример для вторичных  бол листов).
                                                        
   Вид расчета 1-бол листы расчитываются по дням с учетом двойного тарифа.                                                          
   Вид расчета 3-бол листы расчитываются по дням без сучета двойного тарифа.                                                          
   Вид расчета 2-бол листы расчитываются по окладу (час тар ставки) по дням .                                                         
   Вид расчета 11-бол листы расчитываются по часам с учетом двойного тарифа .                                                         
   Вид расчета 13-бол листы расчитываются по часам без сучета двойного тарифа.                                                          


                Справ коэф больничных.
   При входе в данный режим работы на экране высвечивается след окно

     ┌----────────────────────---─┐
     │       Период        │Коэф- │
     │    с     │    по    │фиент.│
     │────────────────────────────│
     │21.08.1995 21.08.1996  2.200│
     │                            │
     │                            │


    Где период означает действие коф т. е. если сумма за месяц для расчета
  больничных попадает в период, то сумма умножается на коэф .
     

        При входе в режим " Выход".
     Hа экране высвечивается след подменю:

           ┌─────────────────────┐
           │  Закончить работу?  │
           ├─────────────────────┤
           │      Нет    Да      │
           └─────────────────────┘
     Выбрав необходимый ответ пользователь закончит работу или продолжить.
